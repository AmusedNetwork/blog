Create a blog post

This page will generate the necessary files for a blog post including the meta JSON file, containing date, title and optionally author .

You can also preview your post within the editor below.

Once you have finished writing your post copy the markdown into a new markdown (.md) file in the posts directory, it's name should be the url slug of your post.

posts / meta / untitled-post.json 
{
    "title" : "",
    "date" : "",
    "author" : false
}
